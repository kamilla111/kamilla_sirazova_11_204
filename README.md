## Структура проекта

- `crawler.py`          — скачивание страниц  
- `process.py`          — токенизация и лемматизация (задание 2)  
- `search.py`           — построение инвертированного индекса + булев поиск (задание 3)  
- `urls.txt`            — список URL для скачивания  
- `pages/`              — скачанные HTML-страницы  
- `tokens/`             — токены по страницам (задание 2)  
- `lemmas/`             — леммы по страницам (задание 2)  
- `inverted_index.json` — инвертированный индекс (задание 3)  
- `index.txt`           — соответствие номера страницы и URL  
- `tokens.txt`          — общий список токенов (для сдачи задания 2)  
- `lemmas.txt`          — общая группировка по леммам
- `tfidf_terms/` — TF-IDF по терминам 
- `tfidf_lemmas/` — TF-IDF по леммам
- `tfidf.py` — расчёт TF-IDF (Задание 4)
- `vector_search.py` - Векторный поиск

## Выполненные задания

### Задание 1 — Краулер
- Скачано 128 страниц на русском языке  
- Обработка ошибок, таймауты, user-agent  
- Создаётся файл `index.txt`

### Задание 2 (Токенизация + Лемматизация)
- Каждой странице соответствует свой файл токенов и лемм
- Использован **pymorphy2** —  лемматизация
- Фильтрация: стоп-слова, короткие слова, мусор, слова с цифрами
- Созданы общие файлы `tokens.txt` и `lemmas.txt` 

### Задание 3 — Инвертированный индекс + булев поиск
- Построен инвертированный индекс → `inverted_index.json`  
- Реализован булев поиск с поддержкой операторов:  
  `AND`, `OR`, `NOT`, скобки `( )`  
- Поддерживаются сложные запросы, например:  
  `(клеопатра AND цезарь) OR (антоний AND цицерон) OR помпей`  
- Интерактивный ввод запросов (до ввода `exit`)

### Задание 4 — TF-IDF

Для каждого из 128 документов подсчитаны TF, IDF и TF-IDF

Отдельно для терминов и для лемматизированных форм

Создано 256 файлов:

– tfidf_terms/1_terms.txt … tfidf_terms/128_terms.txt

– tfidf_lemmas/1_lemmas.txt … tfidf_lemmas/128_lemmas.txt

### Задание 5 — Векторный поиск (Vector Space Model)

- Реализован поиск по TF-IDF леммам
- Используется косинусное сходство
- Ранжирование результатов по релевантности
- Интерактивный режим (как в булевом поиске)
- Скрипт: `vector_search.py`
