 Описание проекта

Данный проект реализует краулер на Python, который:

- Скачивает минимум 100 текстовых страниц из заранее подготовленного списка
- Сохраняет каждую страницу в отдельный файл (.html)
- Создает файл index.txt с соответствием:
  номер файла — ссылка

Структура проекта

- crawler.py — основной скрипт
- urls.txt — список ссылок
- pages/ — папка со скачанными страницами
- index.txt — индекс соответствия

Выполнение задания

В рамках задания:
1. Подготовлен список из 128 страниц на одном языке.
2. Реализован краулер с обработкой ошибок.
3. Скачивание производится без удаления HTML-разметки.
4. Создается index.txt.
