 Deployment Manual

1. Клонирование репозитория

git clone https://github.com/kamilla111/kamilla_sirazova_11_204.git

 2. Установка зависимостей

pip install requests

3. Подготовка списка ссылок
Создать файл urls.txt и добавить туда минимум 100 ссылок

 4. Запуск программы
python crawler.py

5. Результат
- В папке pages будут сохранены html-файлы
- Будет создан файл index.txt
