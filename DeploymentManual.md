# Deployment Manual
Назначение

Проект реализует веб-краулер на Python, который скачивает не менее 100 текстовых HTML-страниц из файла urls.txt и сохраняет их в папку pages. Формируется файл index.txt, содержащий соответствие номера файла и URL.


# Клонирование репозитория
git clone https://github.com/kamilla111/kamilla_sirazova_11_204.git
cd kamilla_sirazova_11_204


# Установка зависимостей

pip install requests

# Подготовка входных данных

В корне проекта должен находиться файл urls.txt:
128 ссылок

# Запуск программы

python crawler.py

# Результат выполнения

После запуска:

создаётся папка pages
сохраняются файлы 1.html, 2.html, …

создаётся файл index.txt формата:
номер_файла ссылка
